# Global-Affair-Exploration
Visualize event data in world map using D3

This is an visualization with full interaction. You can click on the line in the line chart below then the events according to that month will appear. You can see the top 5 country with highest intensity will appear in the left and you can click them to see the detail of that country. Then there is a line chart below able to be clicked. Back to the main page, you can click on the each country on the map, then you can see the arc be filtered and some information will be showen on the left. You can also zoom in and zoom out in the map. Radio button on the left can work, please try it!!!
