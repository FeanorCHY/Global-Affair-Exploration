Group 7: Hongyuan Cui (hoc27)| Rong jie (jir18) | Wenxing Li (wel85)

Introduction

The world is highly connected now, we have events and crisis happens each day. As a member of this world we may want to know how many crisis happens for each country and what account for these crisis. However, there are billions of events and we cannot use traditional way to read them one by one. Therefore, we need an efficient visualization tool to explore the world. In this project, we display the crisis number of each country and international and national events used to explore possible reasons for crisis.

About Data

For the source of data, we have two sets: crisis and events. Crisis dataset contains how many crisis happens for each country in each month. Event dataset contains all recorded events with source, target, time and intensity of it. Here are 5 types of crisis: insurgencies, rebellions, inter-state conflicts, ethnic and religious violence, and domestic political crisis. Here we sum them up into a single value. Events have source and Target which can be decomposed into country and sectors of that country. For example, here is military army of country A attacks country Bâ€™s government. So the source country is A and source sector is military army and target country is B and target sector is government. Of course, events also contain the condition that target country and source country is the same and we can take this as national events. If we want to know the political environment for the world, we do not consider the sectors and we should consider that when we need to explore the political environment solely for a single country. Each event has an intensity value which ranges from -10 to 10 which can evaluate how negative or positive of that event.

Preparation Work

In this visualization, we assume that only negative crisis can lead to crisis, so we remove all positive events. then we aggregate the event data by month, country and sum intensity as international data. Then aggregate the dataset more by sectors if we want national event data of a certain country. There are 2 gigabytes for events dataset and our visualization compresses all of them into several graphs so our visualization if efficient for users to explore the world.
